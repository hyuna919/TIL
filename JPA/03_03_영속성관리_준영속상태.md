### 준영속 상태?
- 영속-> 준영속  
- detached : 영속 상태의 엔티티가 영속속 컨텍스트에서 분리
- 영속석 컨텍스트가 제공하는 기능을 사용하지 못하게 된다


### 준영속 상태로 만드는 방법
- 하나씩
```
member.setName("aaa"); // 상태 변경
em.detach(member);     // detached
tx.commit();            // member는 더이상 JPA가 관리x, 쿼리 안생긴다
```
  
- 영속성 컨텍스트 초기화
```
em.clear();
```

- 영속성 컨텍스트 종료
```
em.close();
```
